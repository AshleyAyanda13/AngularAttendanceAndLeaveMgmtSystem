About Application

This Clock-In Leave Management System Application was Developed By Ayanda Mthombeni. It was developed using Angular 19 for the front end and Springboot for the Back-end.

The Front-end sends API request to the back end. And there are multiple API Endpoints including for Authentication and Authorisation when you login which utilizes Spring Security. All other endpoints cannot be accessed unless you have logged in.
